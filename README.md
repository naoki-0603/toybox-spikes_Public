# 概要
LREngineに組み込むための先行技術検証用プロジェクトです。
RHIの抽象化やマルチスレッドレンダリングなど、低レイヤの技術検証をアジャイルに行うためのサンドボックスです。

## 技術検証
### 描画システム（RHI/マルチスレッディング）
- RHIアーキテクチャとフロントエンド/バックエンドの完全分離
- DOD（データ指向設計）に基づくゼロ・アロケーション戦略
- 完全ロックフリーなマルチスレッド描画基盤（RenderJobSystem）

## ビルド方法
### 必須環境（Requirements)
- OS（Windows 10/11）、コンパイラ（MSVC / VS2022等）
### 依存関係（Dependencies)
- Vulkan SDK、Windows SDK（DirectX用）
### 手順
- VSのバージョンに合わせた SetupVSXXXX.batを実行する。
- 実行後、build-msvcフォルダが生成され、フォルダー内部に生成された.sln(slnx)を開く。
- ビルドを行う。
- ビルド完了後に、ソリューション一覧から RendererApplicationを右クリック→デバッグ→新しいインスタンスの開始を選択。
